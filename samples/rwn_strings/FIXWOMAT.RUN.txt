 This utility program will attempt to find the INVTXN record that 
 corresponds to each WOMAT record, and recalculate the total cost 
 in the WOMAT record (field WOMAT.COST) based on the unit cost of 
 the transaction from the INVTXN record (NOTE: if using FIFO/LIFO 
 costing, this may be the average cost at the time of the trans-  
 action, rather than the actual unit cost).                       
                                                                  
 This program should only be used if you had material issues      
 (using AW-F, AW-G, AW-I, AW-J) prior to 6/6/99 AND use lot or    
 serial control on some or all of your components.                
                                                                  
 Only material-issue type INVTXN records will be considered.  If  
 an exact match cannot be found in INVTXN (based on date, part#,  
 quantity, and WO#), the nearest material-issue INVTXN record     
 (based on date) will be used to at least provide an estimated    
 cost.                                                            
                                                                  
 Please be sure to make a backup copy of your WOMAT.B file, and   
 check the results using JC-M.  Ready to begin?
