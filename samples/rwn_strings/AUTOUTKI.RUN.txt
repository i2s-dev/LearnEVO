 This utility will automatically remove binary zeroes that may           
 potentially be creating posting problems in a number of different       
 areas including posting of sales orders, receiving purchase orders,     
 processing payroll checks, etc.                                         
                                                                         
 Binary zeroes most commonly can creep into department codes and         
 because binary zeroes are not the same as a space character (even       
 though on screen or via maintain database you cannot tell the           
 difference), GL accounts may not be found creating posting amounts      
 that go to the clearing account or worse.                               
                                                                         
 All users will need to be out of certain files before continuing.       
 In the event a particular file that this utility will update is open    
 by someone else, you will be notified.   This utility could take        
 a VERY long time so pick a time when the system will not be in use.
