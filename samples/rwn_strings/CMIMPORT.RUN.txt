CMIMPORT  Import CM AccountsA
This works a lot like the other DBA import programs except:I
1) Goes directly into live BKCMACCT file, no intermediate file, so back up BKCMACCT!A
2) Comma-delimited format only.A
3) If account already exists and you indicated REPLACE, this is essentially a MERGE - only the fields you specify will be replaced, the rest left alone.A
4) You must always specify account code as one of your import fields.A
5) Best of luck and remember #1!A
