JC-S Work Order Detail Report

Purpose of Program

Use this report to get a Detailed  Job Cost  broken down by Labor, Material, Overhead, Outside Processing and Extra cost.

General Program Operation

Enter the filters to identify the Work Order(s) to be included and whether to insert a page break between work orders.  The report will include detailed transaction listings of all Material, Labor, Overhead, Outside Processing and Extra Cost with a subtotal for each and a grand total work order cost.
