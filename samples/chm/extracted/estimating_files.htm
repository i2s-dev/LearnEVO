Estimating Files

 

The Estimating module uses its own set of files for inventory, Bills of Materials and Routings.  The inventory files are a parallel set of the item master files that contain all the production inventory items as well as any new  items added within the Estimating module.  When a new item is added to the production inventory files, it is also added to the Estimating inventory files to keep them synchronized.  New items added to the Estimating Inventory files are not added to the Production inventory files until an Estimate becomes an order.

 

Estimating Bill of Materials contain the Estimate number as well as the parent part number so there can be multiple Bills of Material for the same item.  When creating the BOM for and Estimate, it can be created item by item, copied from a Production BOM or copied from an Estimating BOM.  When copying a BOM, all levels of components are copied along with their routings which can then be edited for this particular estimate.

 

If The BOM is entered manually, Routings can still be copied from production items or other Estimates.  The routing file also contains the Estimate number along with the item number.
