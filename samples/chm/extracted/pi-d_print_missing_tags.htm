PI-D Print Missing Tags

PI-D  Print Missing Tags

Purpose of Program

Use this report to get a listing of any missing tags.  The report looks for and lists any missing tags or ranges of missing tags within the sequence of tag numbers already entered.

General Program Operation

Enter the Year and Phys/Inv No.  The fields default to the latest physical inventory on file.

You can limit the report by selecting only specific Locations (factories or warehouses).

You can start the report at a particular tag number or press  <Enter> at the Starting Tag Number field to get a listing of all missing tags.
