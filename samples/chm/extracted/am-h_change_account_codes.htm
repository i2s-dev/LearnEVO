AM-H Change GL Account Codes

AM-H  Change GL Account Codes

Purpose of program

Use this program to change data from one GL account code to another.  Primarily this is for people who want to re-do their chart of accounts.  It eliminates having to make journal entries from the old account to the new account, and also allows one to easily clear and delete an account.  It can change from an old account to a new account or it can merge two existing accounts together but the "New" account must first be created in AM-C  Enter General Ledger Accounts.

General Program Operation

Enter the Old GL Code and the New GL Code and you will see the program begin processing.

If the new GL account is used in AD-A  General Ledger Defaults, it must be manually changed in that program.
