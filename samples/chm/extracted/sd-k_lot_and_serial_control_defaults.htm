SD-K Lot and Serial Control Defaults

Field Explanations

Print Lot Listing after Invoices/Packing Slips?

If you enter Y to this default, any time you print a graphical format packing slip or invoice that has items with lot control entered, a listing of lot numbers will be included.  For text formats, you will be asked if you want to print a lot number listing.  If you answer yes, a follow-up page will be printed (repeating the order header section) with a listing of the item numbers, quantities, and corresponding lot numbers.

Print Serial Listing after Invoices/Packing Slips?

If you enter Y to this default, any time you print a graphical format packing slip or invoice that has items with serial control entered, a listing of serial numbers will be included.  For text formats, you will be asked if you want to print a serial number listing.  If you answer yes, a follow-up page will be printed (repeating the order header section) with a listing of the item numbers and corresponding serial numbers.
